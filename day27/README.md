# Day 27 - Bias vs Variance Analysis

## Tasks Completed
- Compared training and testing accuracy
- Identified underfitting and overfitting
- Adjusted model complexity using tree depth
- Visualized learning behavior

## Observations
- Low model complexity caused underfitting.
- High model complexity increased overfitting risk.
- Balanced train and test accuracy provided the best generalization.

## Key Learning
A reliable Machine Learning model should balance bias and variance to perform well on unseen data.
# Bias-Variance Analysis

- Low tree depth caused underfitting.
- High tree depth increased overfitting.
- Medium depth provided the best balance.
- Best models generalize well to unseen data.
